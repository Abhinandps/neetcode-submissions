## 📌 What is NeetCode?

[**NeetCode.io**](https://neetcode.io) is your ultimate coding interview prep platform featuring:

- 🎯 **Curated Problems** — systematically organized by topic and difficulty
- 🎬 **Video Solutions** — detailed explanations from industry experts
- 💻 **In-Browser Editor** — code directly without setup hassles
- 🔗 **GitHub Sync** — automatically save your submissions

This repository automatically collects all your NeetCode solutions in one place!

---

## ⚙️ How GitHub Sync Works

### Step 1: Connect GitHub
Visit [neetcode.io/profile/github](https://neetcode.io/profile/github) and link your GitHub account

### Step 2: Choose Your Sync Mode
Pick how you want solutions saved:

| Option | Description |
|--------|-------------|
| **Auto-Commit** | Every submission automatically pushed to this repo |
| **Bulk Sync** | Push all past solutions at once |
| **Manual Sync** | Sync individual submissions from the history panel |

### Step 3: Watch It Happen
Your solutions appear here automatically as you solve problems! 🎉

## 📁 Repository Structure

Solutions are organized by **topic**, then by **problem ID**:

```
📦 neetcode-submissions/
 ┣ 📂 Data Structures & Algorithms/
 ┃ ┣ 📂 two-integer-sum/
 ┃ ┃ ┣ 📄 submission-0.py
 ┃ ┃ ┣ 📄 submission-1.py
 ┃ ┃ └ 📄 submission-2.py
 ┃ ┗ 📂 binary-search/
 ┃   └ 📄 submission-0.ts
 ┗ 📂 Python For Beginners/
   └ 📂 python-hello-world/
     └ 📄 submission-0.py
```

**Key Features:**
- Each problem gets its own folder
- Multiple submissions tracked with versioning
- File extension matches the language used

## ⚡ Quick Settings

Manage your preferences at **[neetcode.io/profile/github](https://neetcode.io/profile/github)**

### 🎛️ Available Options

| Setting | Purpose |
|---------|---------|
| **Auto-Commit Toggle** | Enable/disable automatic commits on new submissions |
| **Status Filter** | Sync all submissions or accepted solutions only |
| **Repository Name** | Rename this repo or start fresh with a new one |
| **Bulk Sync** | Push all historical solutions at once |
